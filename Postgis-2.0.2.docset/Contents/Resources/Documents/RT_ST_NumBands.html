Name

ST_NumBands — Returns the number of bands in the raster object.

Synopsis

integer ST_NumBands(raster rast);

Description

Returns the number of bands in the raster object.

Examples

SELECT rid, ST_NumBands(rast) As numbands
FROM dummy_rast;

rid | numbands
----+----------
  1 |        0
  2 |        3
